Recover Identity

If you have lost access to your identity, you can start the process of recovering it here using the email registered to the identity.

You are prompted for the username or email of the identity you want to recover.

We will send a verification code to the email registered to the identity and you will be prompted to enter this verification code along with the new password.

If entered correctly the password of the identity will be updated and you will be prompted to login with the new password.
